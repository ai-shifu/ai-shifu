# 音频排查日志清理

## 核心规则

- 优先按标记词全局检索（如 `listen-audio-debug`、`音频中断排查`）。
- 删除日志时同步移除仅为日志服务的辅助变量和函数。
- 变更后修正 `useCallback/useEffect` 依赖与未使用变量。

## 工作流

1. 全局检索标记词并列出命中点。
2. 分模块删除日志函数与调用点。
3. 清理关联辅助状态、参数与依赖。
4. 执行定向 ESLint 或类型检查，确认行为不变。
